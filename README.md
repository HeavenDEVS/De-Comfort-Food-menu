A responsive, attractive, and intuitive restaurant menu website showcasing Nigerian dishes across categories: Rice, Soups, Swallows, Snacks, Proteins, Drinks, and Specials.

## Features

- Hero section with brand and playful visuals
- Category chips and keyword search
- Responsive image grid with prices and spice level
- Dish details modal with ingredients and allergens
- Dark mode with persistence (localStorage)
- Accessible semantics (aria roles, live regions, keyboard interactions)

## Getting Started

Just open `index.html` in a modern browser. No build step required.

## Customization

- Edit `script.js` to add, remove, or modify dishes and categories.
- Adjust colors and sizing in `styles.css` under the `:root` and `.light` theme blocks.

## Image Credits
